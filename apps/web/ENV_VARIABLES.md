# Variables d'environnement - Web App

## Configuration requise

Copiez ce fichier vers `.env.local` et remplissez les valeurs.

```env
# API
NEXT_PUBLIC_API_URL=https://api.klozd.com

# App URL
NEXT_PUBLIC_APP_URL=https://my.klozd.com

# Environment
NODE_ENV=production
```
